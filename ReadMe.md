### DrawDiary Project
옛날 그림일기 그리던 시절을 떠올리며 기획된 그림일기 프로젝트 입니다.
그림과 글을 옛날 그림일기 방식으로 만들어내어 소장할 수 있고,
더 나아가 페이스북, 인스타그램, 카카오톡 들에 공유하는 시스템을 도입할 예정입니다.
